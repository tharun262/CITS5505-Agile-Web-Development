# KeepLite

An online note-taking tool that is built upon Google Keep. This software can be used by individuals to capture their thoughts, tasks, and reminders .

## 📌 Description of the Project

KeepLite is a web-based note-taking application which allows users to capture and organize their notes in an efficient way.It is a great tool that facilitates getting organized and sharing information.It provides you with options for creating text and checklist notes that you will be able to modify by coloring and labeling them. Users will also be able to share information to make it available to anyone. This project is being developed as part of the Agile Web Development unit (CITS5505).

## 🎯 Goals

The main goals of this project include:

- Creation of a note-taking app
- Utilizing basic web development principles such as client-server architecture.- Utilization of a database for non-changing data storing
- Allowing users to communicate via shared content
- Using Agile methodology throughout the process 

## 👥 Users to Target

The target audience includes:

- Students to record their notes and assignments
- Individuals planning their daily activities
- Those looking for a simple alternative to sophisticated productivity solutions 

## ⭐ Key Features

### 🔐 Managing Users
- User registration
- Safe login/logout procedures 

### 📝 Managing Notes
- Create notes in text form
- Create your checklist notes
- Modify the existing notes
- Delete the notes 

### 📌 Features of the Organization
- Pin essential notes
- Save notes
- Color the notes
- Label the notes in order to categorize them

## 🔍 Searching and Filtering
- Perform searches for notes based on keywords
- Display notes related to particular labels

## 🌍 Features for Collaboration
- Make your notes public or private
- View other users' shared notes  

## 🏗️ System Architecture

KeepLite has a 

**client-server architecture**:

- ** Client-side (Front-end): **
- Built in Bootstrap, HTML, CSS, and JavaScript.- Takes care of the presentation layer

- ** Back-end (Server-side): ..**

- Developed in Flask (Python)
- Takes care of application logic 

- **Database: **
- SQLite database
- Stores user settings, notes, labels, and information data   

## 🖥️ User Interface Design

The design of the KeepLite's user interface is inspired by Google Keep and relies entirely on simplicity.## Principles of Design:
- Simple layout
- Easy to navigate
- Responsive design.
- Sorted note cards

## Components of UI:
- Search bar to navigate
- Sidebar navigation (labels, archive, etc.)- Cards with notes displayed
- Pinned notes separate from regular notes
